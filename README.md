⚽ Sistema de Sócio Torcedor

Sistema simples desenvolvido em Python para simular um programa de sócio torcedor de um clube esportivo.

Projeto criado para o PIM I da Universidade Paulista utilizando conceitos básicos de programação com Python.

📌 Funcionalidades
Cadastro de sócios
Controle de mensalidade
Simulação de venda de ingressos
Aplicação automática de descontos
Menu interativo no terminal
Bloqueio de acesso para inadimplentes

▶️ Como Executar

Clone o repositório:

git clone https://github.com/seuusuario/seurepositorio.git

Entre na pasta:

cd seurepositorio

Execute o programa:

python main.py
